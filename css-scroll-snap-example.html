CSS Scroll Snap Examples

Example 1: Vertical List

Key properties used:

  • scroll-snap-type: y mandatory - Creates vertical snap points that are required stopping positions
  • scroll-snap-align: start - Elements snap to the top of the container

.vertical-list {
    height: 400px;
    width: 100%;
    overflow-y: scroll;
    
    /* Scroll Snap Properties */
    scroll-snap-type: y mandatory;
}

.vertical-list-item {
    height: 200px;
    scroll-snap-align: start;
}

Example 2: Horizontal Slider with scroll-padding

Key properties used:

  • scroll-snap-type: x mandatory - Creates horizontal snap points that are required stopping positions
  • scroll-snap-align: center - Elements snap to the center of the container
  • scroll-padding: 20% - Adds padding to the snap container, creating an offset for snap points

.horizontal-slider {
    width: 100%;
    height: 300px;
    overflow-x: scroll;
    display: flex;
    
    /* Scroll Snap Properties */
    scroll-snap-type: x mandatory;
    scroll-padding: 20%; /* Creates offset for snap points */
}

.horizontal-slider-item {
    min-width: 70%;
    height: 100%;
    scroll-snap-align: center;
}

Mandatory vs. Proximity Comparison

Compare the behavior difference:

  • scroll-snap-type: x mandatory (above example) - Always snaps to points
  • scroll-snap-type: x proximity (below example) - Only snaps when you're close to a snap point

Try scrolling between items and stop in the middle - the proximity version doesn't force snapping unless you're close to a snap point.
